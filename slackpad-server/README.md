# README

## Setup
```console
$ bin/setup
```

## Getting Started
### Server
```console
$ bin/rails s
```

### Client (debug)
```console
$ yarn run wscat --connect localhost:3000/ws
```

## Run test
```console
$ bin/rails db:reset RAILS_ENV=test
$ bin/rspec
```
