# 今日作るもの

## お題

**チャットアプリ**(e.g. slack)

## 目標

- [ ] ログイン画面があり、名前を入力するとチャットサーバーにログインできる
- [ ] ホーム画面があり、アプリからメッセージを送信できる
- [ ] ホーム画面には過去ログが表示され、メッセージがリアルタイムに更新される

# タイムテーブル

- 10:00 - 14:30 講義形式
- 14:30 - 17:30 自習(発展課題、各自独自機能実装)
- 17:30 - 18:30 3分ずつ発表
- 18:30 - 19:00 日報タイム
